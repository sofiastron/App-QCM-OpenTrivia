Mini-Projet QCM 

## Description

Ce projet  visant à créer un système de gestion de QCM (Questionnaires à Choix Multiples).
Il utilise l’API [Open Trivia Database (OpenTDB)](https://opentdb.com) pour générer dynamiquement des questions de quiz.

L’utilisateur peut configurer le type de quiz qu’il souhaite passer, générer un quiz et visualiser ses résultats.

## Fonctionnalités

 Formulaire de configuration du quiz :

  * Nombre de questions
  * Catégorie
  * Difficulté
  * Type de questions (QCM ou vrai/faux)
  * Codage des caractères
* Génération de quiz personnalisés via l’API OpenTDB
* Affichage des résultats après le quiz
* Interface simple et interactive
  
## Technologies Utilisées

HTML , CSS ,JAVA SCRIPT
**API** : [Open Trivia Database](https://opentdb.com/api_config.php)



## Installation

1. Clonez le dépôt :

git clone https://github.com/votre-utilisateur/mini-projet-qcm.git


2. Ouvrez le dossier dans votre éditeur de code.
3. Lancez le projet dans un navigateur ou via un serveur local (ex. **Live Server** pour VS Code).
